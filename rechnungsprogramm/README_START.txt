RECHNUNGSPROGRAMM STARTEN

1. ZIP entpacken
2. START_APP.bat doppelklicken
3. Beim ersten Start wird node_modules erstellt. Das ist normal.
4. Danach sollte der Browser automatisch öffnen.
5. Falls nicht, im Browser öffnen:
   http://127.0.0.1:5173

Wichtig:
- Das schwarze Fenster muss offen bleiben, solange du die App benutzt.
- Zum Beenden einfach das schwarze Fenster schließen.
- Deine Daten werden lokal im Browser gespeichert.
- Zusätzlich kannst du in der App "Datei speichern" nutzen.

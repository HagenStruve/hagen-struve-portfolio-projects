# LeadPilot

LeadPilot ist ein browserbasiertes Portfolio-Projekt für strukturierte Leadgenerierung, regionale Eingrenzung und Datenexport.

## Start

- `LeadPilot-Scraper-v5/index.html` im Browser oder per VS Code Live Server öffnen.
- Es ist kein Build-Prozess nötig.
- Die App läuft statisch und ist FileZilla-kompatibel.

## Demo-Modus

Der Demo-Modus erzeugt realistische Beispieldaten direkt im Browser. Es werden keine externen Seiten gescraped und keine Schutzmechanismen umgangen.

## Google API vorbereitet

Das API-Key-Feld ist für eine spätere Google-Places-/Maps-API-Anbindung vorbereitet. Der Key bleibt lokal im Browser und wird nicht in das Repository geschrieben.

## Export

LeadPilot kann Leadlisten als CSV oder JSON exportieren. Die exportierten Daten können anschließend einem LLM gegeben werden, um Leads zu sortieren, zu clustern und zu bewerten.

## Hinweis

Die Nutzung echter APIs muss rechtskonform, datenschutzbewusst und nach den jeweiligen API-Nutzungsbedingungen erfolgen.

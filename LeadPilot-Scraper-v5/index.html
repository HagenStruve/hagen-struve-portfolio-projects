Bedienung

So nutzt du LeadPilot

OpenStreetMap aktiv
1. Datenquelle wählen
  • OpenStreetMap: kostenlos, ohne API-Key.
  • Google Places: mit API-Key, bessere Daten.
2. Suche eingrenzen
  • Branche/Keyword eingeben, z. B. Landmaschinen, Werkstatt oder Lohnunternehmen.
  • Bundesland, Kreis/Region und Stadt ergänzen.
3. Leads prüfen
  • Suche starten.
  • Score, Status und Kontaktdaten prüfen.
  • CSV/JSON exportieren oder optional an ein LLM geben.
Google Places API einrichten
  1. Google Cloud Projekt erstellen.
  2. Places API aktivieren.
  3. Billing aktivieren.
  4. API-Key erstellen.
  5. Datenquelle „Google Places“ wählen und API-Key in LeadPilot einfügen.

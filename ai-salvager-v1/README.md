# AI Salvager

Modernes statisches Browsergame-Projekt ohne Build-Prozess.

## Starten

Einfach `index.html` im Browser öffnen oder den gesamten Ordner per FileZilla hochladen.

## Struktur

```text
/assets
/css
/js
/sounds
/images
index.html
```

## Version 1 enthält

- modernen Startscreen
- Fullscreen Canvas Renderer
- responsive Skalierung für Desktop und Mobile
- animierten Space-Hintergrund mit Parallax-Feeling
- steuerbares Salvage-Schiff
- WASD / Pfeiltasten / Maus / Touch
- HUD mit Energie und Score
- Pause-Menü
- modulare JS-Struktur

## Nächster Schritt

Gameplay-Grundlage:
- Sammelobjekte
- Asteroiden
- Kollisionen
- Partikel bei Sammlung
- kleiner Screen Shake

# Codex-Prompt für Licht im Feld

Du arbeitest in einem Vite/React-Projekt namens **Licht im Feld**.

Ziel: Eine ruhige, performante, interaktive Erlebniswebseite über Jesus, Hoffnung, Vergebung und Frieden.

Wichtige Regeln:

- Das Erlebnis darf nicht überladen werden.
- Performance ist wichtiger als zusätzliche Effekte.
- Mobile First: Die Seite muss auf Smartphones weich laufen.
- Die Atmosphäre soll ruhig, hochwertig, filmisch und nicht kitschig bleiben.
- Keine aggressiven Missionsformulierungen.
- SEO/AISO soll strukturiert sein, aber das Erlebnis nicht sichtbar zerstören.
- Bilder liegen in `public/images/`.
- Vor Änderungen Tests ausführen oder ergänzen.
- Nach Änderungen:
  - `npm test`
  - `npm run build`

Mögliche nächste Aufgaben:

1. Performanceprofil auf Mobile verbessern.
2. Bildvarianten für Mobile/Desktop ergänzen.
3. Sound optional erst nach User-Interaktion einbauen.
4. Landingpage + Impressum/Datenschutz ergänzen.
5. OpenGraph final auf echte Domain anpassen.
